Project 1: Movie Database App & API

Two projects in one, this "Bechflix" app that uses a REST API to display movie information to the user as well as allow for certain user actions.

Back-end: "movie_api" - The server-side component of an IMDb-like web application. API is a Node.js & Express application using REST architecture with URL endpoints for all user action features. The database was built using MongoDB, modeled with Mongoose and deployed via Heroku. See GitHub README file for further details on app features.

myFlix API GitHub

Front-end: "myFlix-client" - A client-side application based on a RESTful API that displays movie information and allows users to register/log-in, view movie details, add movies to a Favorites list and updated profile information. Built using React & Bootstrap, along with Redux, Axios & Parcel.

Project 3: Meet App

A 'meet-up' application that uses Google Calendar API to display event details for users in their searched city.

Overview - A serverless PWA built with React using a TDD technique that fetches data from the Google Calendar API and implements OAuth2 authentication for user login. User actions include searching for a specific city, or clicking "see all cities", plus editing how many events are displayed at a time. Also includes two charts to effectively display types of events (pie-chart) & number of events per city (scatterplot).

Project 4: Chat App

A chat app developed using React Native, Expo and Gifted Chat for use on both Android and iOS mobile devices.

Overview - The app includes two screens: Start, a home screen for users to choose a username and background color preference before entering the chat. Chat, the chat interface created with Gifted Chat to send text messages along with images and the user's location. Created using Expo with storage database via Google Firestore (v7).
